### How to run

* clone this directory.
`git clone https://github.com/Meraj-Sharif-Khan/user-management-api.git`

* Install dependencies.
`npm install`

* Add .env variables .
`PORT = `
`MONGO_DB_URI = `
`JWT_SECRET = `
`NODE_ENV = "development"`

* Run by.
`npm start`
